【NOI2019模拟赛（四十六）】宝石魔术

2019-03-20
  • 生成函数

题意简述

你有一些物品，每个物品有一定的价值，你需要支持如下四种操作：

  1. 加入一个价值为xxx的物品
  2. 删除一个价值为xxx的物品（保证之前存在这种物品）
  3. 求有多少种不同的选取方案，使得总价值为kkk，每个物品视作是不同的
  4. 求有多少种不同的选取方案，使得总价值为kkk，价值相同的物品视作是相同的

输入的所有数都不超过10410^410​4​​

题解

本题是退背包的经典问题，一般退背包的问题用生成函数来做会非常简单

对于第一种询问，考虑每个物品选或不选。于是得到生成函数：

∏i(xwi+1)\prod_i(x^{w_i}+1) ​i​∏​​(x​w​i​​​​+1)

插入就是二项式乘法，删除就是二项式除法，询问就是第kkk项系数，暴力做即可

对于第二种询问，考虑价值为wiw_iw​i​​的物品选几个。设这样的物品共有cic_ic​i​​个，于是得到生成函数:

∏i(∑j=0cixwij)=∏ixwi(ci+1)−1xwi−1\prod_i\left(\sum_{j=0}^{c_i}x^{w_ij}\right)=\prod_i\frac{x^{w_i(c_i+1)}-1}{x^{w_i}-1} ​i​∏​​(​j=0​∑​c​i​​​​x​w​i​​j​​)=​i​∏​​​x​w​i​​​​−1​​x​w​i​​(c​i​​+1)​​−1​​

插入或删除时，cic_ic​i​​改变，除以原来的(xwi(ci+1)−1)\left(x^{w_i(c_i+1)}-1\right)(x​w​i​​(c​i​​+1)​​−1)，再把新的(xwi(ci+1)−1)\left(x^{w_i(c_i+1)}-1\right)(x​w​i​​(c​i​​+1)​​−1)乘上即可

比较舒服的是，我们并不需要特判cic_ic​i​​从000变成111的情况。因为对于在插入之前ci=0c_i=0c​i​​=0的情况，我们在上述的操作种恰好会除以一个(xwi+1)\left(x^{w_i}+1\right)(x​w​i​​​​+1)，于是分母就自动到位了。cic_ic​i​​从111变成000也不需要特判，原因类似

因为询问的kkk也不超过10410^410​4​​，所以我们只需在modx104\text{mod}\;x^{10^4}modx​10​4​​​​意义下维护生成函数即可。复杂度O(q×104)O(q\times10^4)O(q×10​4​​)

完
